# Uday Diwane Portfolio

A storytelling-driven Data Analytics portfolio showcasing my journey from a Diploma in Information Technology to building data-driven projects using Python, SQL, Power BI, Tableau, Machine Learning and Data Visualization.

## Live Website



## About

This portfolio is designed as an interactive journey rather than a traditional resume website. Each section represents a chapter of my growth:

* The Beginning
* The Toolkit
* The Projects
* The Future

The goal is to showcase not only the tools I have learned, but also the problems I have explored and the insights I have discovered through data.
